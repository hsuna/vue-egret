## Introduction

VueEgret 主要是应用在 egret 引擎库上的，它诞生的初衷是为了 egret 支持 Vue 的开发方式，在 egret 上所实现的一套 MVVM 框架。

## Ecosystem

| 项目                                                          | 描述                                           |
| ------------------------------------------------------------- | :--------------------------------------------- |
| [vue-egret-loader](https://github.com/hsuna/vue-egret-loader) | 基于 webpack 的单文件组件(\*. .vue 文件)加载器 |
